## Descripción de la Solución
En este trabajo lo que hicimos fue un programa para ordenar números. Usamos Java y tratamos de hacerlo con Programación Orientada a Objetos para no dejar todo el código en un solo lugar. La idea era entender bien cómo funcionan los algoritmos, por eso hicimos QuickSort y MergeSort nosotros mismos y no usando algo ya hecho. Así se entiende mejor cómo se van organizando los datos paso a paso.

También se separó el programa en varias clases para que no quedara tan enredado y cada parte hiciera lo suyo.

## ¿Que hace?

1. Recibe un arreglo de números
2. Lo ordena con QuickSort
3. Lo ordena con MergeSort
4. Muestra cómo estaba antes y cómo queda después
5. Funciona con diferentes casos (números normales, negativos o ya ordenados)

## Guía de Instalación

### Requisitos
- Tener instalado Java JDK 11 o superior
- Contar con todos los archivos .java del proyecto en la misma carpeta

### Archivos necesarios
- ArraySorter.java
- QuickSorter.java
- MergeSorter.java
- Main.java

### Ejecución por consola (CMD)
1. Descargar el proyecto desde GitHub
2. Descomprimir el archivo ZIP en una carpeta
3. Abrir la carpeta donde están los archivos del proyecto
4. En la barra superior del explorador de archivos, escribir cmd y presionar Enter
<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/7be40954-8f5f-4b1e-b1c5-4531378d3916" />

6. Se abrirá una terminal ubicada directamente en esa carpeta (lista para ejecutar Java)
  
7. Compilar el programa con el comando: javac Main.java
   
8. Ejecutar el programa con el comando: java Main

<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/1ecf44b8-7e12-4add-98d9-0569a46d58b4" />

### Ejecución en Visual Studio Code
1. Tener visual studio code
2. Instalar la extensión Java Extension Pack
3. Abrir la carpeta del proyecto en visual studio code
4. Verificar que todos los archivos .java estén en la misma ubicación
5. Abrir el archivo Main.java
6. Ejecutar el programa con el botón Run
